¿Qué es VSC? - Concepto

Visual Studio Code es un editor de código fuente desarrollado por Microsoft para Windows, Linux y macOS. Es un editor de código fuente súper rápido y liviano que se puede usar para ver, editar, ejecutar y depurar código fuente para aplicaciones. Es utilizado principalmente por desarrolladores front-end. Visual Studio Code es un producto completamente diferente en comparación con Visual Studio.

Caracteristicas

Visual Studio Code es una herramienta que tiene soporte nativo para gran variedad de lenguajes, entre ellos podemos destacar los principales del desarrollo Web: HTML, CSS, y JavaScript, entre otros. Otra ventaja interesante es la posibilidad de configurar la vista a nuestro gusto.

Proceso de instalación

  1. Paso 1: Ve a la página de Visual Studio en Academic Software y solicita tu código de licencia haciendo clic en el botón Solicitar.

  2. Paso 2: Haz clic en el botón 'Descargar Visual Studio Enterprise 2019' a la derecha para descargar el software.

  3. Paso 3: Elige qué software adicional quieres descargar y haz clic en Install. La instalación se iniciará.
